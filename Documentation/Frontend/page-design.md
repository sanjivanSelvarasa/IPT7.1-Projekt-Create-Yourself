    - Wie funktionierts?

    - Login
    - Registrieren

## Footer
- Links:
    - Impressum
    - Datenschutz

## Home-Page
```
Home Page
│
├── Navbar
├── Hero Section
├── Was ist Create Yourself?
├── Features
├── So funktioniert's
├── CTA Section
├── Vertrauen / Vorteile
└── Footer
```

### Hero Section
- Headline
- Subheadline
CTA Buttons
    - Jetzt Starten (Registrierung)
    - Demo ansehen

### Was ist Create Yourself?
- 2-Spalten:
    - (Links) Kurze Beschreibung
    - (Rechts) Beispiel Portfolio / Vorschau

### Features Section
- 3 Cards nebeneinander:
    - Baukasten Editor
    - Einfache Bedienung
    - Individuelles Design

### So funktioniert's
- 3 Schritte Anleitung
    1. Registrieren
    2. Portfolio erstellen
    3. Veröffentlichen & teilen

### CTA Section
- Text zum CTA führt
- Button "Jetzt registrieren" (Kontrast)

### Vertrauen / Sicherheit
- Sicherer Login (JWT etc.)
- Schnelle Ladezeiten
- Einfache Bedienung
